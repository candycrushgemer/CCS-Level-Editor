Global User Display System Examples

Avatar Display

Using data-user-avatar attribute:

<div data-user-avatar class="avatar-example"></div>

Using data-user-avatar attribute on IMG:

<img data-user-avatar width="100" height="100" style="border-radius: 50%;">

Using .data-user-avatar class:

<div class="data-user-avatar avatar-example"></div>

JavaScript API Examples

Manual refresh:

// Refresh all user displays userDisplay.refresh(); // Or use the direct function refreshUserDisplay();

Get current user data:

// Get user data object const userData = userDisplay.getData(); console.log(userData); // Check if user is logged in if (userDisplay.isLoggedIn()) { console.log('User is logged in'); }

How to Include

Add this to your HTML pages after Firebase SDK:

<!-- Firebase SDK --> <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-app-compat.js"></script> <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-auth-compat.js"></script> <script src="https://www.gstatic.com/firebasejs/9.0.0/firebase-firestore-compat.js"></script> <!-- Your Firebase config --> <script src="credentials.js"></script> <!-- Global User Display System --> <script src="global-user-display.js"></script>

Available Attributes and Classes

Type Usage Result
data-user-avatar Attribute Sets src (IMG) or background-image (DIV)
.data-user-avatar Class Sets src (IMG) or background-image (DIV)
data-user-email Attribute Sets textContent
.data-user-email Class Sets textContent
data-user-username Attribute Sets textContent
.data-user-username Class Sets textContent
